# 大语言模型（LLM）基础

大语言模型（Large Language Model，LLM）是一种基于深度学习、通过海量文本训练而成的语言理解与生成模型。
它的核心架构是 Transformer，由 Google 在 2017 年提出，通过自注意力机制（Self-Attention）高效建模长距离依赖。

LLM 的训练分为两个阶段：预训练（Pre-training）和微调（Fine-tuning）。
预训练阶段，模型阅读数万亿 token 的互联网文本，学习语言的统计规律和世界知识。
微调阶段（包括指令微调 SFT 和基于人类反馈的强化学习 RLHF），让模型学会遵循指令、进行对话。

LLM 的核心能力是"下一个 token 预测"——给定上文，预测最可能的下一个词。
这个简单目标在海量数据上训练后，涌现出了推理、翻译、写代码、问答等复杂能力。

LLM 的局限性包括：训练数据有截止日期（无法知道最新信息）、容易产生幻觉（编造不存在的事实）、
无法访问私有数据（企业内部文档、数据库）、上下文窗口有限（通常 128K tokens）。
这些局限正是 RAG（检索增强生成）要解决的问题——通过外部检索注入实时、准确的知识。
