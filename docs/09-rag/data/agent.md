# AI Agent 概念

AI Agent（人工智能智能体）是一种能够自主感知环境、做出决策并执行行动以完成目标的系统。
一个完整的 Agent 通常包含三个核心环节：感知（Perception）、决策（Reasoning）、执行（Action）。

在现代大语言模型（LLM）的语境下，AI Agent 通常指以 LLM 为大脑、结合外部工具和记忆系统、
通过多步循环（Agent Loop）自主完成复杂任务的系统。它不是单轮问答，而是能持续运转的自主实体。

Agent 的关键能力包括：工具调用（调用搜索、计算、API 等）、记忆（跨轮次保留上下文）、
推理（ReAct 模式：Thought → Action → Observation）和规划（把大任务拆成子步骤）。

Agent 与普通聊机器人的本质区别在于"自主性"：聊天机器人被动回答问题，而 Agent 主动规划、
主动检索信息、主动纠错，直到完成任务。多 Agent 系统则让多个 Agent 分工协作，各自负责规划、执行、审查等不同角色，共同完成更复杂的工程任务。
