**目标**：使用更智能的情绪分析。

**任务**：
1. 不仅检测关键词，还要检测情绪强度（轻微不满 vs 极度愤怒）

## 练习5：添加对话摘要

**目标**：当对话过长时自动压缩记忆。

**任务**：
1. 在 ConversationBuffer 中添加 `summarize()` 方法
2. 当消息数超过 20 条时，自动调用 LLM 生成对话摘要
3. 用摘要替换旧消息，保留最近 5 条完整消息

**提示**：
```python
def summarize(self) -> str:
    """调用 LLM 生成对话摘要。"""
    prompt = "请将以下对话总结为一段简洁的摘要：\n\n"
    for msg in self.messages[:-5]:
        prompt += f"{msg['role']}: {msg['content']}\n"
    return llm_chat(prompt, "你是一个对话摘要助手。")
```
